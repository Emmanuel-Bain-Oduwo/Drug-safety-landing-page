# Drug Safety Landing Page

Static frontend for the MedSafe Kenya medication safety landing page.

## Deploy on Vercel

Set the Vercel project Root Directory to:

```text
frontend
```

No build command is required. Vercel can serve `index.html` directly.

## Files

- `index.html` - page markup and SEO metadata
- `styles.css` - responsive UI styling
- `script.js` - interactive checklist
- `assets/` - optimized hero images, social preview image, and icons
- `vercel.json` - static deployment headers and clean URLs
